# Module 2: Business Friction to Use Case

## 🎯 Purpose
Teach the learner to identify genuine business pain points and translate them into a bounded, scored, and documented AI use case before writing any prompts or code.

## 📦 Expected Output
- **[AI Opportunity Brief](templates/ai-opportunity-brief.md)**
- **[Use Case Scorecard](templates/use-case-scorecard.md)**

## 📚 Lessons Outline
1. How to identify business friction and bottlenecks.
2. How to map user intent, workflow context, and success boundaries.
3. How to score AI use cases based on feasibility, value, and risk.
4. How to write a robust AI Opportunity Brief.
5. Lab: Scoping your Capstone project candidates.
