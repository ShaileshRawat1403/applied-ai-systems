# Module 4: Knowledge Systems and RAG

## 🎯 Purpose
Teach how to organize unstructured corporate knowledge, set up document ingestion parameters, design metadata filtering, and ground AI systems using Retrieval-Augmented Generation.

## 📦 Expected Output
- **[RAG Readiness Checklist](templates/rag-readiness-checklist.md)**
- **[Retrieval Design Template](templates/retrieval-design-template.md)**

## 📚 Lessons Outline
1. Unstructured data audit: chunking sizes, semantic boundaries, and text hygiene.
2. Indexing and metadata strategy (filtering context by category/author/security).
3. Retrieval pipelines: vector vs. keyword vs. hybrid searches.
4. Grounding prompts: instructing models to cite sources and handle missing facts.
5. Lab: Designing a grounding and context retrieval harness.
