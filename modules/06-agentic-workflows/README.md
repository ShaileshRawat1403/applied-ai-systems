# Module 6: Agentic Workflows and Orchestration

## 🎯 Purpose
Design multi-step workflows, configure system state, map router-orchestrator loops, establish memory retention limits, and manage token context size.

## 📦 Expected Output
- **[Agent Workflow Canvas](templates/agent-workflow-canvas.md)**
- **[Memory and Context Policy](templates/memory-context-policy.md)**

## 📚 Lessons Outline
1. Agents vs. Directed Workflows: when to choose control over autonomy.
2. Orchestration patterns: sequential chain, routing classifier, and critic-reviewer.
3. State management: maintaining session values and database persistence.
4. Memory optimization: summaries, buffer truncations, and context pruning.
5. Lab: Designing a critic-reviewer loops canvas for automated drafting.
