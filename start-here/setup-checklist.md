# Setup Checklist

Before you begin Module 0, complete this checklist to ensure your local technical workspace is configured correctly.

---

## 📋 Technical Setup Requirements

### 1. Version Control
* `[ ]` Git is installed on your local system (`git --version`).
* `[ ]` You have initialized a local Git repository in your course folder:
  ```bash
  git init
  ```
* `[ ]` Create a basic `.gitignore` file to avoid committing keys or build logs (see the [Workspace Setup Guide](../modules/00-orientation-and-setup/02-workspace-setup.md) for details).

### 2. Markdown Editor & Previews
* `[ ]` Your editor supports Markdown files and syntax highlighting.
* `[ ]` You have a viewer that can render **Mermaid.js** diagrams (e.g., VS Code with Mermaid extensions, Obsidian, or GitBook Preview).

### 3. API Playground Accounts (Optional but Recommended)
* `[ ]` Initialize an API key with your chosen LLM vendor:
  * Google AI Studio (Gemini API keys)
  * Anthropic Developer Console
  * OpenAI Developer Portal
* `[ ]` Verify that you can successfully run a simple test API call in your terminal (using `curl` or a Python/Node script).

---

## 🏁 Verification
Once these steps are marked complete, proceed to [Module 0: Orientation and Course Setup](../modules/00-orientation-and-setup/README.md) to select your use case and create your learning log.
